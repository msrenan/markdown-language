# Links

Finally, to end the basics of the Markdown language, this tutorial will teach how to put local_links to your markdown.

Why would we do that? Good question! Imagine that you want to connect a piece of information in one file to a whole paragraph that is in another one. This would be a great use of links.

## Tips

So, the basic form of a link is: `[PLACEHOLDER](path)`, lets have an example:

```
[MY LINK](./images/blobs.png)
```

[MY LINK](./images/blobs.png)

> The image usage we learned on the previous tutorial is a specific use of links, thats why sintax is similar.

> Images and GIFs uses a **macro** to display its content on rendering (the macro is signed by `!`)

The sintax of links is all you need to build some cool things. Lets hava a better look.
