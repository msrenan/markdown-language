# Images

On this part of the tutorial you will learn how to use images and GIFs in your markdown files!

## Tips

So let's think you want to display a screenshot or a diagram of your project/repository. Markdown lets you do that by using a simple sintax:

```
![IMAGE](./images/blobs.png)
```
![IMAGE](./images/blobs.png)

> Do not forget the `!` before de `[`, otherwise it will become a **link** which we will discuss on the last basic tutorial.

It is also possible to put some movement to your .md texts adding GIFs instead of images, using similar sintax:

```
![GIF](./images/coolgif.gif)
```

![GIF](./images/coolgif.gif)

> Using images and GIFs in markdown texts is basically this. There will be no example for this tutorial.
